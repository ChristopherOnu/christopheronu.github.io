Speak Digital — Automated Client Onboarding System

ZapierWorkflow AutomationDigital AgencyOperations
Built a fully automated client onboarding system for Speak Digital Marketing Agency that handles new client intake across three service tiers: Basic, Standard, and Premium. From the moment a client submits an intake form, the system checks for duplicates, routes the workflow based on the service package selected, creates the relevant project folders, assigns tasks, and sends personalised welcome communications to both the client and the appropriate internal team, all without any manual involvement.
The Problem

Speak Digital was onboarding new clients manually across three different service packages. Every new submission meant someone had to check whether the client already existed in the system, create the right folders, set up project tasks, send a welcome email tailored to the package, and notify the relevant team. Doing this correctly for every client across Basic, Standard, and Premium tiers was time-consuming, prone to inconsistency, and pulled the team away from actual delivery work every time a new client signed up.

The Solution

Built a multi-path Zapier workflow triggered by a Jotform intake form. The moment a client submits, the system searches the existing client database to check for duplicates. If the client exists, their record is updated. If not, a new record is created. Client name and date fields are formatted and cleaned automatically before any further steps run. A lookup table maps each service tier to the correct team lead and Google Drive folder. From there, the workflow splits into three distinct paths based on the service tier selected. Each path handles folder creation, task setup, client email, and Slack notification in a way that matches exactly what that tier requires, with Premium clients receiving the most comprehensive onboarding sequence and Basic clients receiving a streamlined minimal flow.

Key Features
  • Jotform intake form triggering the full workflow instantly on submission
  • Duplicate check running on every submission, updating existing records and creating new ones only when needed
  • Automatic data formatting splitting client names into first and last, cleaning text fields, and formatting dates before they are used anywhere in the workflow
  • Lookup table mapping each service tier to the correct team lead email and Google Drive folder destination
  • Premium path creating a dedicated client folder in Google Drive, setting up a full project with multiple tasks including brand discovery call, strategy document, design kickoff, and review meeting, sending a detailed welcome email with timeline and next steps, and notifying the Premium team lead on Slack with full client details
  • Standard path creating a client folder, setting up focused project tasks including onboarding call, content calendar, and first post review, and notifying the Standard team lead via Slack and Gmail
  • Basic path adding the client to a shared tracking spreadsheet, sending a simple welcome email, and firing a brief Slack notification to the Basic team with no additional steps
  • Clean conditional logic ensuring each path runs only what is needed for that tier with no unnecessary steps
Tools Used
Zapier Jotform Google Drive Gmail Slack
Results & Impact

Speak Digital eliminated the entire manual onboarding process for new clients across all three service tiers. What previously took up to 45 minutes per Premium client, covering CRM checks, folder creation, project setup, and email communication, now completes in seconds automatically. At five new clients per month, that recovers several hours of team time that previously went into administrative onboarding rather than delivery. Every client receives a consistent, professional experience from the moment they sign up, and every team lead gets an immediate Slack notification with the details they need to get started.
